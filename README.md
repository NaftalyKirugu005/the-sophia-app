# The Sophia App
developed june of 2026
## Overview
The Sophia App is a student management application developed using Android Studio. The application allows users to:

- Login to the system
- Register students
- Save records locally
- Navigate between screens

## Features
- User Authentication
- Student Registration
- Local Data Storage
- Simple Navigation
- Responsive User Interface

## Technologies Used
- Kotlin
- Android Studio
- SQLite/Room Database
- Material Design Components

## Installation
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the application.
